Start Bootstrap | Framework CSS

Comment fonctionne un framework css ?

Un framework, c’est une base d’outils et de fonctionnalités conçus pour s’agencer entre eux.
En partant d’une grille, qui est la plupart du temps prévue pour être responsive, vous pouvez ajouter facilement des éléments dont la plupart des sites ont besoin: des comptes utilisateurs, des authentifications, des connexions à diverses sources de données, des moteurs de recherche interne, des boutons de partage sur les réseaux sociaux…
Par ailleurs, le framework est automatiquement mis à jour pour s’adapter aux nouvelles pratiques du net comme ce fut le cas pour le passage aux sites responsive.

Qu'est-ce que "Vanilla JavaScript" ?

Le terme script vanilla est utilisé pour désigner le JavaScript pur (ou nous pouvons dire le JavaScript brut) sans aucun type de bibliothèque supplémentaire.
Parfois, les gens l'ont souvent utilisé comme une blague "de nos jours, plusieurs choses peuvent également être faites sans utiliser de bibliothèques JavaScript supplémentaires".
